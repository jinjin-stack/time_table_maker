      ⚙️ GitHub 자동 저장 설정

      owner/repo는 jinjin-stack/time_table_maker 로 이미 박혀 있습니다.
      PAT 1회만 입력하면 이 브라우저에서 자동 저장이 동작합니다.

      PAT 발급:
      1. github.com/settings/personal-access-tokens/new
      2. Repository access → time_table_maker 1개만 선택
      3. Permissions → Contents = Read and write
      4. 생성된 github_pat_… 복사 → 아래 입력
      ⚠️ PAT는 이 브라우저 localStorage에만 저장됩니다 (공용 PC에서 작업 후 비우세요)
